===============标准包使用须知==============

1.注意！！！git仓库名需与主文件名保持一致。
  该标准包中的HelloWorld.js为主文件，需将该文件重命名为：git仓库名.js。

2.file目录存放项目相关的资源文件。

3.可将该标准包上传到git仓库中使用。

===============开发须知====================

1.修改HelloWorld.js中场景地址 为 离线开发网络版中上传的场景地址，如下所示：

var app = new THING.App({
    url: 'http://服务器IP:端口号/MMDScene/场景名',  // 场景地址
    skyBox: 'BlueSky',
	resourceLibraryUrl: "./"
});

2.更多内容请查看“离线开发网络版-用户手册”。
